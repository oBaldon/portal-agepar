Este guia define **como** escrever exemplos de código de forma consistente.

## Padrões
- Sempre usar `title="..."` nos blocos que representam arquivos ou comandos importantes.
- Preferir `showLineNumbers` quando o snippet tiver mais de ~10 linhas.
- Destacar linhas críticas com `{...}` ou **magic comments**.
- Comando ≠ Saída: separar em blocos distintos.
- Preferir abas (`Tabs/TabItem`) para múltiplas linguagens.
- Usar `diff` para mudanças, e `mermaid` para diagramas.

## 3) Destaque de linhas (2 jeitos)

**a) Por intervalo:**

```python title="apps/bff/main.py" {3,5-7}
from fastapi import FastAPI
app = FastAPI()
@app.get("/api/me")                  # linha 3 destacada
def me():                            # linhas 5-7 destacadas
    return {"ok": True}
```

**b) Por “magic comments”:**

```ts title="fetch.ts"
const r = await fetch("/api/me");
// highlight-next-line
if (!r.ok) throw new Error("Auth required");
/* highlight-start */
const data = await r.json();
renderUser(data);
/* highlight-end */
```

## 4) Blocos de diferença (diff)

```diff title="apps/bff/app.py"
- app = FastAPI()
+ app = FastAPI(title="Plataforma AGEPAR BFF")
```
