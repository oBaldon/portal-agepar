## Estrutura principal

```text
apps/docs-site/
  docs/
  blog/
  src/
  static/
  docusaurus.config.ts
  sidebars.ts
  package.json
```

## Pontos importantes do estado atual

- `baseUrl: "/devdocs/"`
- navbar aponta para a sidebar técnica
- o Host proxia `/devdocs` para `http://docs:8000`
- o projeto ainda contém `package-lock.json` e `pnpm-lock.yaml`

## Sidebar

A sidebar principal usa `autogenerated` por diretório, então novos arquivos
dentro das seções numeradas entram automaticamente na navegação.
